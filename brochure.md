# Project Architect

**Start every project like a pro.**

---

## What It Does

Project Architect is a project scaffolding tool that generates complete, professional-grade project structures with a single click. Answer a few questions about what you're building, and it creates everything: source code skeleton, documentation suite, testing framework, Git setup, and AI-ready context files—all following proven best practices.

## The Problem It Solves

Starting a new project right takes hours. You need folder structures, README files, requirements.txt, testing setup, linting configuration, Git initialization, and all those little files that make a project *professional*. Most developers skip half of it and regret it later.

Project Architect front-loads that work. Every project you create starts with the same solid foundation—consistent structure, complete documentation, quality tools already configured. No more "I'll add tests later" or "I'll write the README when I'm done."

## Key Features

- **Interactive Wizard** — Answer questions about your project, get a complete scaffold tailored to your needs
- **Multiple Templates** — Desktop GUI (CustomTkinter, PyQt, Tkinter), Web Apps (Flask, FastAPI, Django), CLI Tools (Click, Typer, argparse)
- **Full Documentation Suite** — Auto-generates README, CHANGELOG, Technical Spec, Implementation Plan, and more
- **AI-Friendly Structure** — Includes AI_CONTEXT.md so AI assistants understand your project instantly
- **Quality Tools Built-In** — pytest, Black, flake8, MyPy—configured and ready to use
- **Git Integration** — Initializes repository with proper .gitignore and initial commit
- **Update System** — Revisit projects later to add features or modify specs without starting over

## Who It's For

- **Solo developers** who want professional project structure without the setup time
- **Teams** who need consistent project organization across multiple projects
- **Educators** teaching best practices in project organization
- **AI-assisted developers** who want their AI tools to understand project context immediately

## Platform Support

- Windows
- Linux
- macOS

## Getting Started

1. Run `python main.py` to launch Project Architect
2. Click "Start New Project" and choose a template
3. Answer the questionnaire about your project requirements
4. Select an output directory
5. Click "Generate" and start coding

Your new project includes a working application, virtual environment, documentation, and quality tools—all ready to go.

## Technology

Built with Python and CustomTkinter. Uses Jinja2 templates for flexible project generation. All output is plain files—no lock-in, no proprietary formats.
